## Bugs Fixed

### B1 - Join codes not generated for new quizzes (P0) ✅

**Root Cause:** `CreateQuizUseCase` wasn't generating or passing a `joinCode` to the Quiz constructor.

**Files Changed:**
- `src/lib/join-code.ts` - NEW: Utility to generate codes in format `JOIN-XXXX`
- `src/application/use-cases/create-quiz.use-case.ts` - Import and use generator

**Implementation:**
```typescript
// src/lib/join-code.ts
export function generateJoinCode(): string {
  // Excludes ambiguous characters: I, 1, O, 0
  const chars = 'ABCDEFGHJKLMNPQRSTUVWXYZ23456789';
  const suffix = Array.from(
    { length: 4 },
    () => chars[Math.floor(Math.random() * chars.length)]
  ).join('');
  return `JOIN-${suffix}`;
}
```

**Tests Added:**
- `src/tests/lib/join-code.test.ts` - 3 tests for format, character exclusion, uniqueness
- `src/tests/application/use-cases/create-quiz.use-case.test.ts` - 3 tests for join code integration

### B2 - Supabase subscription errors flooding console (P1) ✅

**Root Cause:** In `subscribe()`, the callback logged an error for ANY status that isn't 'SUBSCRIBED', including the normal intermediate state 'SUBSCRIBING'. This caused 50+ console errors during normal operation.

**File Changed:**
- `src/infrastructure/realtime/supabase-realtime-client.ts`

**Fix:** Only log errors for actual error states, not intermediate states:
```typescript
// Before: if (status !== 'SUBSCRIBED') { logError() }
// After:
if (status === 'CHANNEL_ERROR' || status === 'TIMED_OUT' || status === 'CLOSED') {
  logChannelIssue('error', 'Supabase subscription error', { ... });
}
```

Also fixed the `emit()` function which had the same issue - it was rejecting for any non-SUBSCRIBED status including 'SUBSCRIBING'.

### B3 - Timer value missing on quiz detail page (P2) ✅

**Root Cause:** DTO shape mismatch. The admin page expected `quiz.timePerQuestion` (from `QuizListItemDTO`) but the API returns `QuizDTO` which has `quiz.settings.timePerQuestion`.

**File Changed:**
- `src/app/(admin)/admin/quizzes/[quizId]/page.tsx`

**Fix:**
- Changed import from `QuizListItemDTO` to `QuizDTO`
- Updated `quiz.timePerQuestion` to `quiz.settings.timePerQuestion`
- Updated `quiz.questionCount` to `quiz.questions.length`

---

### B5 - `/host` route returns 404 (P2) ✅

**Root Cause:** No `page.tsx` existed in `src/app/(host)/` directory. Home page links to `/host` but route didn't exist.

**File Created:**
- `src/app/(host)/page.tsx`

**Implementation:** Simple redirect to admin quizzes page:
```typescript
import { redirect } from 'next/navigation';

export default function HostPage() {
  redirect('/admin/quizzes');
}
```

---

### B6 - HTML nesting warnings in player view (P3) ✅

**Root Cause:** Invalid HTML nesting - a `<div>` (ScoringInfoBadge container) was inside a `<p>` tag in the player join form.

**File Changed:**
- `src/components/player/player-join-form.tsx`

**Fix:** Moved the ScoringInfoBadge `<div>` outside of the `<p>` tag:
```tsx
// Before: <p>Text... {condition && <div>...</div>} </p>
// After:
<p>Text...</p>
{condition && <div>...</div>}
```
